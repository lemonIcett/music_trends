  This project, "Global Music Trends and Popularity Analytics," presents a comprehensive Power BI dashboard analyzing 114,000 tracks from the Spotify Track Audio Features and Genre Dataset sourced from OpenDataBay. The analysis explores five key objectives: identifying AI-driven popularity drivers using the Key Influencers visual, profiling genre-specific audio characteristics through heatmap matrices, evaluating artist performance via aggregated bar charts, examining song density across danceability and energy dimensions using treemaps, and analyzing tempo trends through dual-axis visualizations. Key findings reveal that low acousticness (≤0.504) and vocal-heavy tracks significantly boost popularity, while slow tempo songs achieve higher average popularity than fast-paced ones despite medium tempo tracks showing peak danceability. The project employs a flat-table schema with DAX measures for dynamic calculations, featuring interactive slicers for genre and explicit content filtering. Future enhancements include predictive machine learning models for pre-release popularity forecasting, personalized recommendation engines based on listener behavior, and enterprise-scale deployment with social media integration and automated refresh capabilities for music industry stakeholders.
